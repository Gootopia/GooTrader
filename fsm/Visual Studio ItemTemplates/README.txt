Use this to add a finite state machine item to the Visual Studio templates list.
This allows you to create a new template for this.
The class will be named whatever the file name is, so for consistency, it is recommended to name it of the form: "FSM_<YourStateMachineName>.cs".

To use, place the .zip file in the visual studio item templates list.
This is found at: %USERPROFILE%\Documents\Visual Studio <version>\Templates\ItemTemplates.
